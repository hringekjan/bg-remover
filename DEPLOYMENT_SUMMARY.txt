================================================================================
DEPLOYMENT FLOW
================================================================================

Dual-Write Pattern (All Parallel):

  writeOutcome(outcome)
    ├─→ Promise.allSettled([
    │     writeMem0(outcome),                    [HTTP, 3 retries, BLOCKING]
    │     writeDynamoDB(outcome),                [AWS SDK, BLOCKING on carousel-main]
    │     writeLcpApi(outcome)                   [HTTP, 2 retries, FIRE-AND-FORGET]
    │   ])
    │
    ├─→ Extract results & detect discrepancy
    ├─→ logDualWritePattern() → CloudWatch Logs
    └─→ emitMetrics() → CloudWatch Metrics (individual + dual-write-complete)

LCP-API Non-Blocking Write Path:

  writeLcpApi(outcome)
    1. Attempt POST to LCP_API_BASE_URL/learning/outcomes/sale
       - Headers: Authorization (Bearer token), x-tenant-id, x-outcome-id
       - Retry: 2 times with backoff (50ms → 100ms)

    2. On success: return { success: true }

    3. On failure: queue to SQS DLQ for async retry
       - Fallback async processing
       - Exponential backoff (1s → 2s → 4s → 8s → 16s)
       - Max 5 retry attempts
       - Then send to dead-letter queue (DLQ-of-DLQ) for manual review

Result: carousel-main response latency UNCHANGED (all writes parallel, LCP-API non-blocking)

PRE-DEPLOYMENT (DevOps):
  1. Create SSM parameters:
     aws ssm put-parameter --name /tf/dev/platform/bg-remover/lcp-api-auth-token \
       --value "<LCP_API_TOKEN>" --type SecureString

     aws ssm put-parameter --name /tf/dev/platform/bg-remover/lcp-api-dlq-url \
       --value "<SQS_DLQ_ENDPOINT>" --type SecureString

  2. Create SQS DLQ:
     Queue: bg-remover-lcp-api-dlq-dev
     Visibility timeout: 300s
     Retention: 14 days

  3. Verify LCP-API is running

DEPLOYMENT:
  cd services/platform/carousel/organisms/bg-remover

  # Validate configuration
  npx serverless print --stage dev > /tmp/config.json
  cat /tmp/config.json | jq '.provider.environment | {LCP_API_BASE_URL, LCP_API_AUTH_TOKEN, LCP_API_DLQ_URL}'

  # Deploy
  npx serverless deploy --stage dev --region eu-west-1 --param="enableCloudWatchMetrics=true"

  aws lambda get-function-configuration --function-name bg-remover-dev-processWorker --region eu-west-1
